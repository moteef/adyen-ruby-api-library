Using the library

Set up a client and set the credentials to the AdyenOfficial platform. 

require 'adyen'

adyen = AdyenOfficial::Client.new

The example connects you to the AdyenOfficial test platform. If you want to use our live platform use: 

adyen.env = :live

You can now use the following services:

  • Checkout
  • Payments
  • Recurring
  • Payouts
  • Marketpay
